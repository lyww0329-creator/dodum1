# 돋움과외 정적 페이지 완성본

이 폴더 전체를 GitHub 저장소 루트에 업로드하세요.

## 생성 범위
- 지역 기본 페이지: 시·도, 시·군·구, 읍·면·동
- 지역+과목 페이지: 국어, 영어, 수학, 사회, 과학
- 지역+학년 페이지: 초등, 중등, 고등, 성인/N수생
- 과목 대표 페이지
- 학년 및 세부 학년 페이지
- 학년+과목 페이지
- 영어·일본어·중국어 회화 페이지
- 상담 신청 페이지

## 중요: 실제 도메인 반영
현재 canonical, sitemap, robots에는 `https://YOUR_DOMAIN`가 들어 있습니다.
도메인이 확정되면 프로젝트 전체에서 이 문자열을 실제 주소(예: `https://dodoomtutor.com`)로 일괄 교체하세요.

## 파일 수
- 정적 URL: 29,638개
- 사이트맵: 3개 + sitemap index 1개

## GitHub 업로드
1. ZIP 압축을 풉니다.
2. 내부 파일과 폴더 전체를 저장소 루트에 업로드합니다.
3. GitHub Pages를 활성화합니다.
4. 실제 도메인을 연결한 뒤 위 placeholder를 교체합니다.
